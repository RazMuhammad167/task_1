🚀 Dart & OOP Fundamentals for Flutter Development
This repository contains a collection of sample command-line applications and library code designed to master the core pillars of Dart and Object-Oriented Programming (OOP).

This project was developed as part of the Mobile App Development Course at NAVTTC (National Vocational and Technical Training Commission), specifically focusing on high-performance coding and data handling.

🎓 Certification & Coursework
Institution: NAVTTC (National Vocational and Technical Training Commission)

Specialization: Flutter Mobile App Development

Key Achievement: Completed advanced module on Encoder Byte processing and data serialization in Dart.

🏗 Project Structure
Plaintext

🧠 Core Concepts Covered
1. Dart Basics
Strong Typing: Utilizing String, int, double, and bool.

Sound Null Safety: Writing secure code using ?, !, and late keywords.

Collections: Mastering List, Map, and Set.

2. Object-Oriented Programming (OOP)
This project demonstrates the four pillars of OOP essential for Flutter:

Encapsulation: Using private variables (_variableName) and getters/setters.

Abstraction: Creating abstract classes to define blueprints for UI components.

Inheritance: Extending classes to reuse code (e.g., class User extends Person).

Polymorphism: Overriding methods to change behavior in child classes.

3. Encoder Byte & Data Processing
A specialized section dedicated to the Encoder Byte concepts learned at NAVTTC:

Converting Strings to UTF-8 Byte Arrays.

Handling binary data for local storage and network requests.

Implementing custom encoders for data obfuscation and serialization.
